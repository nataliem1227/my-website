Treemaps

Python Pygame Pytest Recursion File I/O OS Library

// OVERVIEW

Built an interactive Python treemap tool using recursive algorithms to visualize hierarchical data (e.g., file systems, research papers) as proportional rectangles.

This project implements a dynamic visualization system that transforms complex hierarchical structures into intuitive visual representations, allowing users to understand data relationships and proportions at a simple glance.
