AI Invoicing System

Typescript React Native Golang Graphql Jira Confluence Expo RAG

// OVERVIEW

AI-driven financial assistant with Golang backend and RAG-based chatbot supporting natural-language CRUD operations, batch invoice processing, and automated agent-to-human workflows for invoicing and scheduling.

// DEVELOPMENT

As the frontend team lead for the CRUD and invoicing system, my primary role was designing and fixing the mobile Figma workflows. I also implemented this functionality in the final stage of the web app.

For my backend role, I built a RAG pipeline for editing an invoice, creating the tool and adding batch editing/update invoices functionality.
