Paint Application

Java JavaFX Ollama Design Patterns Regex

// OVERVIEW

Developed a desktop paint application using JavaFX and Model-View-Controller architecture. Implemented Strategy, Command, and Factory design patterns for extensible shape drawing tools and file persistence with regex-based finite state machine parsing.

Integrated Ollama AI for automated paint file generation through natural language prompts in a collaborative Git-based Scrum environment.
